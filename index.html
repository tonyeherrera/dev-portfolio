  • The Cocktail Library

    This project is my favorite by far. This web application allows visitors to view drink recipies with the help of a third party api. Recipes are searchable by name, ingredient, or alcoholic/non-alcoholic. Recipes can also be added to a list and saved in local storage for later viewing.

    There is a user authenticated portion of the site ("Library Card") where visitors can signup to view recipe submissions from other users and add their own. This portion of the site is still under construction.

    The Cocktail Library was developed using React.js for front-end rendering, Node.js for backend management, and MongoDB Atlas for data storage. Token authentication wasa used for the user account portion of the site ("Library Card".)
